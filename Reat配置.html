⚛️ React 零基础快速通关秘籍

从纯小白到成功运行第一个 React 网页的完整心法与指南

🗺️ 核心学习路线图

第一阶段（前置）：不要急着碰 React，掌握 JS 的变量、箭头函数、结构赋值以及数组 map() 方法。

第二阶段（筑基）：聚焦于 JSX 语法、组件化开发（函数组件）、Props 传参和核心的 useState 与 useEffect Hooks。

第三阶段（避坑）：现阶段忽略 Oxlint、Webpack 或高级框架，顺其自然使用脚手架中自带的 ESLint 规范代码即可。

🛠️ 第一步：两大家底安装

  1. Node.js：前往官方网站 nodejs.org 下载左侧的 LTS 长期支持版。
  2. VS Code：下载并完成常规安装。
验证安装：打开终端输入 node -v 和 npm -v，能打印出版本号即为成功。

🎨 第二步：VS Code 精英级配置

1. 推荐安装的三大插件：

  • Chinese (Simplified) - 简体中文包。
  • ES7+ React/Redux/React-Native snippets - 快捷键神器（输入 rfce 秒出组件模板）。
  • Prettier - Code formatter - 代码一键美化。

2. 开启保存自动格式化：

按 Ctrl + , 打开设置 ➔ 搜索 format on save ➔ 勾选 Editor: Format On Save。

🚀 第三步：终极解决方案（解决 npm 下载慢）

如果在 npm install 阶段卡死或极慢，请在 VS Code 终端里顺序执行以下国内阿里云镜像切换命令：

# 1. 切换为阿里云淘宝镜像源
npm config set registry https://registry.npmmirror.com

# 2. 检查是否切换成功（应该返回刚刚设置的阿里云网址）
npm config get registry

🔥 第四步：正规军建项目流程（杜绝路径报错）

  1. 在 E 盘新建空文件夹 ReactProjects。
  2. 用 VS Code 点击 文件 ➔ 打开文件夹 选中它。
  3. 点击菜单 终端 ➔ 新建终端（确保开局路径精准为 E:\ReactProjects>）。
  4. 在终端里依次敲入并执行以下命令：
# ① 一键创建 React 骨架
npm create vite@latest my-react-app -- --template react

# ② 终端走入刚刚建好的项目口袋里
cd my-react-app

# ③ 下载 React 全部源码配件（速度已通过上一步起飞）
npm install

# ④ 燃火启动本地服务器！
npm run dev
💡 Windows 脚本运行报错拦截？
如果弹出红字“禁止运行脚本”，点击终端右上角 + 号旁边的下拉箭头，切换成 Command Prompt (cmd) 终端重新粘贴命令，即可完美绕过！

🎉 见证奇迹：修改你的第一行 React 代码

当终端输出绿色网址 http://localhost:5173/ 时，按住 Ctrl 键点击它打开浏览器。

接着在 VS Code 左侧找到 src/App.jsx 文件，将里面的 return(...) 的内容清空，修改为以下极简代码并保存：

function App() {
  return (
    <div style={{ textAlign: 'center', marginTop: '50px' }}>
      <h1>⚛️ 我的第一个 React 网页启动成功！</h1>
    </div>
  )
}
export default App

回到浏览器，网页已经实时刷新，属于你的 React 旅程正式开启！
